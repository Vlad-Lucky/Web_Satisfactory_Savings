Описание проекта Web
Название игры/проекта: “Satisfactory Savings”

Проект представляет из себя сайт, основанный на flask, который помогает реализовать следующую задачу:
требуется реализовать ресурс, в котором будет хранится файл сохранения игры Satisfactory; к этому сохранению
будут иметь доступ все зарегистрированные пользователи, и они могут загрузить себе сохранение, чтобы
поиграть (для этого нужно открыть доступ по сети, а на сайте указать id для подключения к другу. К сожалению,
сайт не будет тестировать, правильно ли открыл пользователь доступ подключения к нему другим). После окончания игры
пользователь отправляет файл сохранения обратно на сайт.

Скорее всего регистрация будет основана на логине дискорда и проверке, есть ли пользователь на дискорд сервере
(если нет, то он не сможет получить id игры для подключения и не сможет загрузить себе сохранение)

В каждый файл сохранения будет вписываться закодированный порядковый номер для того, чтобы нельзя было загрузить
прошлое или совершенно другое сохранение.

Планируется добавить меню "поддержка проекта", через которое пользователь сможет поддержать проект денюжкой.
