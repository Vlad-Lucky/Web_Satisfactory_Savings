Техническое задание.

1. Идея
Идея игры базируется на проблеме развития сохранения всем друзьям вместе, не прибегая к дорогостоящим "выделенным серверам"

2. Работа с файлом сохранения Satisfactory
2.1 Реализовать удобную возможность вписывания закодированного порядкового номера в файл сохранения игры так, чтобы
игра смогла запуститься без проблем
2.2 Реализовать удобную возможность проверки порядкового номера сохранения

3. Сайт
3.1 Регистрация
Пользователь вводит логин дискорд-акканта, пароль и обычный логин
3.2 Авторизация
Пользователь вводит (логин дискорд-акканта + пароль) или (обычный логин + пароль)
3.3 Скачивание сохранения (на этом этапе по идее должна будет идти проверка, есть ли данный дискорд-аккаунт на дискорд-сервере)
3.4 Указывание данных об открытой сессии (если какая-то сессия уже открыта, то другой пользователь не может открыть
вторую по тому же сохранению)
3.5 Загрузка сохранения на сайт
3.6 Меню поддержки проекта денюжкой (будет реализовано, если qiwi api не будет заблокирован)

4. Дискорд-бот
4.1 Бот пока планируется только для получения логинов дискорд-аккаунтов на сервере. Он будет запускаться вместе с сайтом.
